  • 2012 - present : P.I., GEMS (Geostationary Environment Monitoring Spectrometer)
  • 2012 - 2020 : Commissioner, International Ozone Commission (IO3C)
  • 2010 - present : Member, CEOS (Committee on Earth Observing Satellites) AC-VC
